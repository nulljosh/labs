# labs

Consolidated small/experimental projects, each in its own subfolder with full original git history.

- [`canlii-app`](./canlii-app) — CanLII case lookup app
- [`missing-pets`](./missing-pets) — Craigslist-style board for posting and finding lost/found pets
- [`books`](./books) — Curated book rankings based on Goodreads ratings and reviews, with a library due-date tracker
- [`wiretext`](./wiretext) — Unicode wireframe design tool — click, place, export as text
- [`grapher`](./grapher) — Data visualization and charting utility
- [`nyc`](./nyc) — Times Square Factory Sim: Factorio-style production game (SpriteKit, Swift 6, iOS + macOS)
- [`nimble`](./nimble) — Native macOS instant-answer search with offline math & query classification

Each subfolder was merged via `git subtree`, so `git log --follow <subfolder>` still shows the original commit history. The original repos are archived (read-only) on GitHub.
